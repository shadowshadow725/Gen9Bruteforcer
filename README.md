# Gen 9 Raid Pokemon BruteForcer 

This plugin is for pkhex can generate legal pokemon in raids for pokemon S/V 

## Usage 

Make a folder called plugins in the Pkhex folder and put the dll in it. 
The menu should appear in tools when pkhex is relaunched

1. Load a save file (Important)
2. Have some junk pokemon in your box 1 for the program to override 
3. Open the Gen9BruteForcer menu in pkhex under tools 
4. Make sure to move all any pokemon you want out of box 1. The program will override everything in box 1 !!!!
5. Set the stats and pokemon you want to bruteforce
6. Start the program and wait
